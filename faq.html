01

加入服务器

如何加入 Minaro？

使用 Minecraft Java Edition 1.21.11，连接 mc.minaro.cc。

需要安装 Mod 吗？

不需要。Minaro 基于 Purpur/插件端，玩家使用原版客户端即可进入。

服务器是否需要白名单？

如果服务器处于内测或小规模阶段，可能会启用白名单。具体加入方式请以公告为准。

02

世界与玩法

Minaro 是什么类型的服务器？

Minaro 是一个小型长期生存世界，重视建筑、铁路、港口、钓鱼、酿酒、聚落与玩家合作。

这是 Hardcore 模式吗？

不是 Hardcore 模式。但死亡、距离、物资与交通都会保留一定代价。

适合什么玩家？

适合喜欢慢生存、建筑、交通系统、长期建设、小社区与世界沉淀的玩家。

03

规则与保护

有地皮插件吗？

没有大范围地皮插件。Minaro 希望世界保持连续开放，但会通过容器锁和 CoreProtect 保护私人物品与建筑。

可以建大型自动农场吗？

不鼓励极端工业化。影响服务器性能、破坏经济或过度刷资源的设施可能会被限制。

有 /tpa 或 /home 吗？

原则上不开放或严格限制。距离是 Minaro 世界设计的一部分，铁路、道路与港口因此才有意义。

04

经济与货币

服务器有金币或余额系统吗？

Minaro 倾向使用实体物品作为货币，而不是传统数字余额。交易会真实发生在地图中。

为什么用毒马铃薯和鹦鹉螺壳？

它们是可流通、可运输、可储存的实体货币，可以让贸易、仓储与运输变成世界的一部分。

汇率固定吗？

可以有官方参考汇率，但玩家之间允许自由议价。不同地区也可能形成不同价格。

05

墓碑机制

死亡会掉落物品吗？

死亡后会原地生成墓碑，玩家需要返回死亡地点取回遗物。

掉进岩浆或虚空怎么办？

插件会尽量在最近安全位置生成墓碑，具体效果以服务器实际设置为准。

有 /back 吗？

不开放 /back。死亡后的回收、救援和交通应成为世界的一部分。

06

技术问题

连接不上服务器怎么办？

请确认版本是否为 Java 1.21.11，并检查地址是否为 mc.minaro.cc。如果仍无法连接，请查看公告或联系管理员。

地图打不开怎么办？

地图服务可能会随服务器维护而暂时关闭。请稍后重试，或查看首页公告。

为什么网站内容没有切换语言？

请尝试刷新页面。如果仍然无效，可能是浏览器缓存或脚本尚未更新。
